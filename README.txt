RUS



СПИСОК ОСНОВНЫХ ФАЙЛОВ

    "/src/Test maker.htm" - генератор тестов

    "/*.htm" - тесты, сделанные с помощью генератора тестов
    
    
    
СПИСОК ДОПОЛНИТЕЛЬНЫХ ФАЙЛОВ

    "/src/*.htm" - дополнительные информационные сведения
    
    "/src/pictures/" - общие картинки, необходимые для тестов
    
    "/src/sounds/" - общие звуки, необходимые для тестов
    
    "/src/*/" - звуки, необходимые для конкретных тестов
    
    
    
ТЕСТЫ

    Необходимо выполнить указанные задания в тесте, заполня поля Answer и
    нажимая кнопку Check. Необходимо соблюдать формат ответа, указанный в
    первом задании и следовать рекомендациям в таблице в поле Additional Info.
    Правильным ответом для каждого задания будет являться конкретная строка,
    именно для этого следует соблюдать формат и рекомендации.
    
    
    
ГЕНЕРАТОР ТЕСТОВ

    Необходимо заполнить иформацию в полях в графе Parameters. Далее при
    необходимости заполнить Intro table. После этого в графе Questions нужно
    для каждого задания выбрать файл со звуком (необходимо указать папку со
    звуками в графе Parameters) и указать правильный ответ для задания. Потом
    следует нажать кнопку Generate и получившийся текст скопировать в новый
    текстовый файл с именем "*.hmt".
